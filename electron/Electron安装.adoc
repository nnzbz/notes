== 创建Demo项目
. 进入要创建项目的上层目录
. 创建项目
+
[,shell]
----
npm init electron-app@latest my-app -- --template=vite-typescript
----
. 进入项目根目录
+
[,shell]
----
cd my-app
----
. 初始化项目
+
[,shell]
----
yarn
----

== 设置国内镜像
. 进入项目根目录
+
[,shell]
----
cd my-app
----
在项目根目录新建 `.npmrc`，内容：
[,shell]
----
electron_mirror=https://npmmirror.com/mirrors/electron/
----

== 安装Electron
. 进入项目根目录
+
[,shell]
----
cd my-app
----
. 安装
+
[,shell]
----
yarn add -D electron
----

== 运行
. 进入项目根目录
+
[,shell]
----
cd my-app
----
. 运行
+
[,shell]
----
yarn start
----

== 编译
. 进入项目根目录
+
[,shell]
----
cd my-app
----
. 编译
+
[,shell]
----
yarn make
----

== 发布
. 进入项目根目录
+
[,shell]
----
cd my-app
----
. 发布
+
[,shell]
----
yarn publish
----
