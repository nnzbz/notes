== 下载最新版dify
[source,bash]
----
cd ~/opt
git clone https://github.com/langgenius/dify.git
----

== 进入docker目录修改配置
[source,bash]
----
cd ~/opt/dify/docker
sed -i 's/langgenius\/dify-api:1.12.0/langgenius\/dify-api/g' docker-compose.yaml
sed -i 's/langgenius\/dify-web:1.12.0/langgenius\/dify-web/g' docker-compose.yaml
----

== 进入docker目录复制配置文件并修改暴露端口

[source,shell]
----
cd ~/opt/dify/docker
cp .env.example .env
sed -i 's/EXPOSE_NGINX_PORT=80/EXPOSE_NGINX_PORT=12080/g' .env
sed -i 's/EXPOSE_NGINX_SSL_PORT=443/EXPOSE_NGINX_SSL_PORT=12443/g' .env
----

== 启动

[source,shell]
----
docker compose -f ~/opt/dify/docker/docker-compose.yaml up -d
----
