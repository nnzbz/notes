## 5. 终端

### 5.1. 快速呼出终端

1. 创建自动操作
 打开 ```Automator.app``` -> ```新建``` -> ```服务``` -> ```服务收到``` 选择为 ```没有服务``` -> 双击左侧的 ```Run AppleScript```

2. 编写自动操作的AppleScript
 在上一步的编辑区输入内容如下：

 ```AppleScript
 on run {input, parameters}
     tell application "Terminal"
         reopen
         activate
     end tell
 end run
 ```

3. 保存
 上一步的内容保存为 ```Open Terminal```

4. 设置快捷键
 系统偏好设置 -> 键盘 -> 快捷键 -> 左边栏中选择 ```服务``` -> 右边列表中找到 ```Open Terminal``` 并勾选
