== 安装
<https://docs.docker.com/compose/install/>

* Deepin/Ubuntu
+
[,shell]
----
apt update
apt install docker compose-plugin
----
* CentOS
+
[,shell]
----
yum update
yum install docker compose-plugin
----

== docker compose.yml
=== version
- 目前为止有三个版本分别为 `1`, `2`, `3`
- 版本越高支持越多的指令
- 区分 `1` 和 `2`（Compose 1.6.0+，Docker Engine 1.10.0+）

== 启动服务
进入到 `docker compose.yml` 文件所在目录，执行下面的命令
[,shell]
----
docker compose -f xxx.yml up -d
----

== 重新构建服务
进入到 `docker compose.yml` 文件所在目录，执行下面的命令
[,shell]
----
docker compose -f xxx.yml up -d --force-recreate
----
[NOTE]
====
- 在compose文件中配置 `stop_grace_period`，指定停止服务时的等待最长时间，默认为 `10s`
====

== 停止所有服务
[,shell]
----
docker compose down
----

== 外部网络
- 创建外部网络 `rebue`
+
[source,shell]
----
docker network create rebue
----
- 在compose文件中配置 `networks`，指定外部网络，默认为 `default`
+
[source,yaml]
----
networks:
  default:
    name: rebue
    external: true
----

== 限制日志大小
- 在compose文件中的单个服务的节点下，配置 `logging`，指定日志大小，默认为 `-1`
+
[source,yaml]
----
  logging:
    options:
      max-size: 8m
----
